# java-calculator-precourse

## 문자열 덧셈 계산기

- 문자열을 입력받아 구분자에 따라 숫자를 추출하고, 덧셈한 결과를 반환
- 기본 구분자 : 쉼표(,) , 콜론(:)
- 커스텀 구분자 : 입력 시 맨 앞에 "//"와 "\n"사이에 넣어 적용
- 유효하지 않은 입력 시 `IllegalArgumentException` 발생 후 어플리케이션 종료

<br>

## 입출력 요구 사항

**입력**

- 구분자와 양수로 이루어진 문자열

**출력**

- 덧셈 결과

```bash
결과 : 6
```

**실행 결과 예시**

```bash
덧셈할 문자열을 입력해 주세요.
1,2:3
결과 : 6
```
